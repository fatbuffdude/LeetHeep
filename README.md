# LeetHeep
Repository to dump files used in leetcode

A structured collection of my solutions, notes, and patterns from ongoing LeetCode practice.  
This repository documents my problem‑solving process as I prepare for technical interviews and strengthen my foundations in data structures and algorithms.

This repo serves as:

  •  A learning journal to track my progress
  •  A reference library of patterns and approaches
  •  A public record of consistent algorithm practice
  •  A portfolio signal for recruiters and hiring managers

My goal is to write clean, readable, and well‑reasoned solutions — not just code that passes.
